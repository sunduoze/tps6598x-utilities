Warning

The FTDI dynamic-link library that is included with the debugging environment will cause a system exit if it does not establish a link with the FTDI USB Windows driver

In addition to executing the FTDI USB Windows driver installer, it is necessary to connect the "USB to Many" FTDI-based access board in order for Windows to load the driver. Those users who do not have an FTDI board must avoid configuration using the FTDI driver or the GUI will exit.
